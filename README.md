## Data Platforms

Platforms for aggregating, managing, and analyzing agricultural data.

- [Climate FieldView](https://climate.com/fieldview/) — Digital agriculture platform for field data analysis and insights.
- [Granular](https://granular.ag/) — Farm management and analytics platform for operational decision-making.
- [FarmLogs](https://farmlogs.com/) — Data-driven farm management and analytics platform.
- [Agworld](https://www.agworld.com/) — Collaborative farm data platform for planning and record-keeping.
- [OneSoil](https://onesoil.ai/) — Platform for precision farming using satellite data and analytics.

## Geospatial & Remote Sensing

Tools for analyzing satellite, aerial, and UAV-based agricultural data.

- [Google Earth Engine](https://earthengine.google.com/) — Platform for planetary-scale geospatial analysis.
- [Sentinel Hub](https://www.sentinel-hub.com/) — Access and processing for satellite imagery and geospatial data.
- [QGIS](https://qgis.org/) — Open-source geographic information system for spatial analysis.
- [ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview) — Comprehensive GIS platform for mapping and analysis.
- [OpenDroneMap](https://www.opendronemap.org/) — Open-source toolkit for processing aerial imagery from drones.

## Sensors & IoT Data

Platforms and tools for collecting real-time agricultural data from sensors.

- [John Deere Operations Center](https://operationscenter.deere.com/) — Platform for machine and field data collection.
- [Arable](https://www.arable.com/) — Sensor-based platform for weather, crop, and environmental data.
- [CropX](https://cropx.com/) — Soil sensor system with analytics for irrigation optimization.
- [Semios](https://semios.com/) — IoT platform for crop monitoring and pest management.
- [Libelium Waspmote](https://www.libelium.com/iot-solutions/smart-agriculture/) — IoT sensor platform for agriculture.

## Weather & Climate Data

Sources of weather and climate data relevant to agriculture.

- [NOAA Climate Data](https://www.noaa.gov/) — Weather and climate datasets for environmental analysis.
- [OpenWeather](https://openweathermap.org/) — API for weather data and forecasts.
- [Meteostat](https://meteostat.net/) — Historical weather and climate data for analytics.
- [NASA POWER](https://power.larc.nasa.gov/) — Agroclimatology data for renewable energy and agriculture.
- [WorldClim](https://www.worldclim.org/) — Global climate datasets for ecological modeling.

## AI & Predictive Analytics

Tools and platforms for applying machine learning to agricultural data.

- [TensorFlow](https://www.tensorflow.org/) — Machine learning framework for predictive modeling.
- [PyTorch](https://pytorch.org/) — Deep learning framework for building AI models.
- [H2O.ai](https://www.h2o.ai/) — AI platform for predictive analytics and machine learning.
- [FastAI](https://www.fast.ai/) — Deep learning library focused on ease of use.
- [Scikit-learn](https://scikit-learn.org/) — Machine learning library for data analysis.

## Farm Analytics & Decision Support

Tools for turning data into actionable agricultural insights.

- [Cropio](https://cropio.com/) — Platform for crop monitoring and yield prediction.
- [Taranis](https://taranis.ag/) — AI-powered agronomic intelligence platform.
- [Farmers Edge](https://farmersedge.ca/) — Data-driven agriculture platform for precision farming.
- [Prospera](https://www.prospera.ag/) — AI-based crop monitoring and analytics platform.
- [Agremo](https://agremo.com/) — Drone-based analytics for agriculture.

## APIs & Data Infrastructure

APIs and infrastructure for accessing and integrating agricultural data.

- [Agromonitoring API](https://agromonitoring.com/api) — API for weather, satellite, and soil data.
- [FarmStack](https://farmstack.co/) — Open data exchange platform for agriculture.
- [FIWARE](https://www.fiware.org/) — Open-source platform for smart applications, including agriculture.
- [Open Ag Data Alliance (OADA)](https://openag.io/) — Open standards for agricultural data interoperability.
- [CKAN](https://ckan.org/) — Open-source data management system for datasets.

## Datasets

Public datasets for agricultural analysis and research.

- [FAOSTAT](https://www.fao.org/faostat/) — Global agricultural data from the UN FAO.
- [USDA Data](https://www.usda.gov/) — U.S. agricultural datasets and statistics.
- [Kaggle Agriculture Datasets](https://www.kaggle.com/datasets) — Community datasets for agriculture and farming.
- [CGIAR Data Portal](https://data.cgiar.org/) — Agricultural research datasets.
- [Ag Data Commons](https://data.nal.usda.gov/) — USDA platform for agricultural datasets.
